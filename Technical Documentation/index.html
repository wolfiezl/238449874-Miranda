What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It defines the structure and layout of a web document by using a system of tags and attributes.

HTML forms the backbone of every web page, providing the structure for adding text, images, links, and other elements.

What is CSS?

CSS (Cascading Style Sheets) is used to style and design the layout of web pages. It controls the visual presentation of HTML elements such as fonts, colors, and positioning.

With CSS, you can create visually appealing web pages by adjusting the look and feel, while HTML defines the content.

HTML Structure

HTML documents are structured hierarchically with nested elements. At the top level, the document starts with the `` declaration followed by the `` element. The document is split into two main sections: the `` and the ``.

Example of basic HTML structure:

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>
<h1>My First Heading</h1>
<p>My first paragraph.</p>
</body>
</html>
HTML Elements

HTML elements are the building blocks of web pages. Each element is defined using opening and closing tags, with content in between. Some common elements include:

  • <h1> - Heading 1 (used for the main heading)
  • <p> - Paragraph (used for text content)
  • <a> - Anchor (used for links)
  • <img> - Image (used for embedding images)
CSS Basics

CSS styles are applied to HTML elements by using selectors. You can select HTML elements based on their tag name, class, id, or attributes, and apply styles to them using various properties.

Example of a basic CSS rule:

h1 {
  color: blue;
}
CSS Selectors

CSS selectors target HTML elements to apply styles. Some common types of selectors include:

  • element - Targets all elements of that type (e.g., p for all paragraphs).
  • .class - Targets elements with the specified class attribute.
  • #id - Targets an element with a specific id attribute.
CSS Box Model

The CSS box model represents the layout of elements as rectangular boxes. It consists of content, padding, border, and margin. Understanding the box model is crucial for layout design and element spacing.

  • Content - The actual content of the element.
  • Padding - Space between the content and the border.
  • Border - The line surrounding the padding.
  • Margin - The space outside the border, separating the element from others.
Responsive Design

Responsive design ensures that web pages look good on all devices by adjusting the layout based on the screen size. CSS media queries allow you to define different styles for various screen widths.

Example of a media query:

@media (max-width: 600px) {
  body {
    font-size: 14px;
  }
}
CSS Flexbox

Flexbox is a layout model in CSS that allows you to align elements efficiently within a container. It provides an easy way to create flexible and responsive layouts.

Example of a flex container:

.container {
  display: flex;
  justify-content: space-between;
}

CSS Transitions

CSS transitions allow you to change property values smoothly (over a given duration). This can be used for hover effects, animations, and other dynamic changes in style.

Example of a CSS transition:

.button {
  transition: background-color 0.3s ease;
}
.button:hover {
  background-color: red;
}
